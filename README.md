### Technologies

##### Front-End

- Typescript
- React
- MUI Icons

##### Back-End

- NodeJs
- MongoDB

# To Run Project

1. create a `.env` file with a mongoURI

2. cd into server and run

   > npm run dev

3. cd into client and run
   > npm run start
